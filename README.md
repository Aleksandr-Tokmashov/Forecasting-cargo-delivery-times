# Прогнозирование сроков доставки грузов

## Описание задачи
В данной работе задача заключается в прогнозировании фактического срока доставки грузов железнодорожным транспортом на основе исторических данных о перевозках. С точки зрения машинного обучения это задача регрессии, где:
- Целевая переменная: Срок доставки факт (в часах)
- Признаки: характеристики рейсов (расстояние, вес груза, тип вагона, временные метки, логистические показатели)
Для повышения качества модели учитываются различные характеристики рейсов, такие как расстояние, вес груза, тип вагона, временные метки и логистические показатели. В качестве основных метрик качества выбраны MAE (Mean Absolute Error) и R² (Коэффициент детерминации). MAE отражает среднюю абсолютную ошибку в часах, а R² показывает долю объясненной дисперсии.
